# Audit Policy

## Immutable audit

Audit records must not be editable or deletable from SQLAdmin.

## Must audit

- Login/logout.
- Permission changes.
- Kando setting changes.
- RuleSet changes.
- RuleSet activation.
- Admin reprocess/retry.
- Recruiter notes and decisions.
- Sensitive data view/export.
- SQLAdmin sensitive changes.
- AI prompt version changes.
