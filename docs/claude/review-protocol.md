# Review Protocol

Every task completion must include:

```text
STATUS:
FILES_CHANGED:
TESTS_RUN:
ACCEPTANCE_CHECK:
RISKS:
NEXT_RECOMMENDED_TASK:
```

## Self-review checklist

- Did I read the task prompt?
- Did I avoid forbidden changes?
- Did I update docs if contracts changed?
- Did I add tests?
- Did I include migrations if schema changed?
- Did I keep Kando read-only?
- Did I include Persian errors?
- Did I preserve AI assistive-only role?
