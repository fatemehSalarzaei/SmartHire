# Review Checklist

## Product invariants

- [ ] Kando remains read-only.
- [ ] Screening/ranking automatic.
- [ ] Manual runs admin-only.
- [ ] AI assistive only.
- [ ] Language percentage not used.
- [ ] Missing data does not default to reject.

## Backend

- [ ] Service layer used.
- [ ] Error contract implemented.
- [ ] Persian messages present.
- [ ] Alembic migration included.
- [ ] Audit included.
- [ ] Tests included.

## Frontend

- [ ] Server-side pagination/filter/sort.
- [ ] Persian states/errors.
- [ ] Permission states.
- [ ] No business decision logic duplicated.

## Security

- [ ] Sensitive data masked.
- [ ] API key not logged/exposed.
- [ ] SQLAdmin restricted.
