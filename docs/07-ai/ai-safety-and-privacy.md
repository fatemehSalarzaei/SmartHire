# AI Safety and Privacy

## Data minimization

Do not send to AI:

- Kando API key.
- phone number.
- email.
- exact address.
- raw payload.
- unrelated personal data.

Allowed by default:

- job title and requirements.
- about me.
- skills.
- work history titles/descriptions.
- education names/fields.
- language skill levels.
- cover letter.

## External provider

If using external AI provider:

- Provider must be approved.
- Store provider/model/prompt version.
- Mask or remove sensitive data.
- Limit raw output visibility to technical roles.
- Audit all AI runs.

## Bias and sensitive data

AI must not infer or use sensitive attributes unrelated to job suitability.
